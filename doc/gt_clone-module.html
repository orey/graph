gt_clone(graph, rootnode, sideeffect=False, **kwargs)

source code 

Clones a node or a graph

This graph transformation implements several options.

  1. Basic cloning Node g_none, new = gt_clone(None, n)

    'g_none' will stay None. 'new' is totally separated from 'n'.

  2. Cloning node with chaining g, new = gt_clone(None, n, False, option='chain')

    Creates a subgraph with the original node 'n' and a new one 'new' connected by an edge in domain. Note that 'new' is provided to be immediately accessible. Note also that we clone in the past.

    'rootnode -PREVIOUS-> new'

  3. Basic Graph clone new_g, n_none = gt_clone(g, None)

    Creates a new graph, cloned from the previous one.

  4. Graph cloned inside the same graph with links new_g, n_none = gt_clone(g, None, option='same')

    This option creates also BridgeEdge objects inside the graph.

Parameters:
  • graph - Instance of Graph or None
  • rootnode - Instance of Node or None
  • sideeffect - bool. This parameter is not used in this GT
  • kwargs - Dict. See below for options
Returns:
tuple: graph, rootnode. The tuple returned can be reinjected in another graph transformation
