gt_filter_attributes(graph, rootnode, sideeffect=True, **kwargs)

source code 

This graph transformation eleminates useless attributes from a Node or from a whole graph.

Parameters:
  • graph - Instance of Graph. Can be None if 'rootnode' is not.
  • rootnode - Instance of Node. Can be None if 'graph' is not.
  • sideeffect - bool; default is True. If False, a copy is returned.
  • kwargs - Expecting: attributes=["att_name1", "att_name2"] as a mandatory option. Another option, in the case of graph filtering, is the restriction to some types only. When filtering a graph, the filtering can be restricted to a list of Node types by using the following option: applicability=["type1", "type2"] Types in the list are strings, corresponding to the value of the 'type' attribute in Node and Edge classes.
Returns:
tuple: graph, rootnode. The tuple returned can be reinjected in another graph transformation.
